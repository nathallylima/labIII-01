Revisão

1. Cite 3 tags fundamentais do HTML:

html, head e body.

2. Quais tipos de listas existem no html?

Lista não ordenada "ul" e lista ordenada "ol".

3. HTML é uma linguagem de programação? Explique

Não, HTML é uma linguagem de marcação de hipertexto, que define a apresentação e estrutura do conteúdo no site, já linguagem de programação executa ações lógicas.

4. Foi solicitado que você altere o título da página. Qual tag devemos utilizar dentro do "head"?

Devemos utilizar a tag "title".

6. Cite 2 tipos (input type=””) adicionados no HTML 5.

month e week

7. Quais principais diferenças podemos apontar entre HTML 4 e HTML 5?

Suporte para áudio e vídeo e novas tags semânticas como: "article", "header", "nav".

8. Qual a utilidade do DOCTYPE

Informar ao navegador qual versão do HTML está sendo utilizada.

9. Os cabeçalhos podem ir de "h1" até "h?"". Qual diferença?

Até "h6". Servem para organizar a página visualmente por grau de importância, como por exemplo "h1" é utilizado para o título principal, "h2" para subtítulos, já o "h6" é o menos importante.

10. Conteúdos como "p", "img" devem ser inseridos no "head" ou "body"? Explique.

São inseridos no "body" que é onde contém o conteúdo visual da página. "head" possui apenas metadados.

11. Podemos enviar 2 tipos de métodos (Verbos HTTP) no form (formmethod=”?”). Cite os 2 tipos e explique as diferenças.

"GET" e "POST". Utilizando "GET" os dados do formulários são enviados diretamente na URL, ficando visíveis as informações escritas pelo usuário. Já com "POST" os dados ficam ocultos, garantindo mais segurança.
